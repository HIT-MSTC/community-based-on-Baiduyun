项目简介
	为志同道合的人提供一个分享社区
	以及以文件分享为核心的简单社交

所用技术
	Php
	百度云php	SDK

设计思路
	整个社区基于web搭建，有利于各个平台的使用，通过网站后台对api的调用，使用户通过简单操作就能实现下列操作
	
	用户 
		注册
		登陆
		个人信息
		文件管理	对用户文件进行上传、下载、拷贝、搜索、获取缩略图等操作
		文件分享和文件的获取
	
	社区
		文件的分类
		文件信息的交流
		文件的评论
		  类似论坛社交形式
	  文件的排名
